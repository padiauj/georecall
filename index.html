GeoRecall

GeoRecall is a map recall game (similar to Seterra): we’ll prompt you with a name and you’ll click the matching region on the map. To get started quickly, select a preset below.

What do these fields mean?
  • Relation ID: OSM relation ID for an area (e.g., campus boundary). Find it in the OSM URL.
  • Subtype key: Tag key to filter features, e.g., building.
  • Overpass endpoint: API to query OSM data. Default is the public instance.
  • Center/Zoom: Optional. If omitted, the map fits to the data bounds.
